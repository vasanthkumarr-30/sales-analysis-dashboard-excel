# 📊 Sales Analysis Report Dashboard (Excel Project)

## 📌 Project Overview
This project is an interactive **Sales Analysis Dashboard** built entirely in **Microsoft Excel** to analyze sales performance, customer behavior, product performance, and order trends.

The dashboard provides business stakeholders with actionable insights through KPIs, charts, and dynamic filters to support data-driven decision-making.

---

## 🎯 Project Objectives
The objective of this project is to:

- Monitor overall sales performance.
- Track revenue trends over time.
- Analyze customer purchasing behavior.
- Identify top-performing products and cities.
- Understand revenue contribution by occasion and category.
- Provide an interactive dashboard experience using slicers and filters.

---

## 📂 Dataset Information

The project uses three datasets:

### 1️⃣ Raw Data Set
Contains the original transactional sales data.

### 2️⃣ Cleaned Data Set
Contains processed and cleaned data used for dashboard creation.

### 3️⃣ Dashboard File
Contains:
- Pivot Tables
- Pivot Charts
- KPI Cards
- Slicers
- Dashboard Design
